## About

### Description and Purpose

The purpose of our application will be to design an intuitive website that will allow us to show case the use of get, post, patch, and delete routes via a RESTful API while using React Router that will gracefully handle errors. The application will allow a user to add their favorite music to the page and then it will be analyzed to find similar music that they may be interested in.

### Inspiration

The inspiration is Spotify and Pandora.
